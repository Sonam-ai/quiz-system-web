# Interactive Quiz System

A web-based quiz app built with Flask, featuring interactive design and software engineering questions.

## Features
- Multiple-choice questions on software engineering topics.
- Progress bar and confirmation before submission.
- Responsive design with CSS and JavaScript.

## How to Run Locally
1. Install Python and Flask (`pip install flask`).
2. Run `python app.py`.
3. Open `http://127.0.0.1:5000` in your browser.

## Deployment
Deploy to Heroku or similar for a live version.
